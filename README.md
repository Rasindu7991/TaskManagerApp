## TaskManager API
The TaskManager API is a RESTful service that handles data storage and retrieval for the application. It supports all CRUD operations for tasks and user authentication. Key API endpoints include:

- `POST /api/tasks`: Create a new task
- `GET /api/tasks`: Retrieve a list of tasks
- `GET /api/tasks/:id`: Retrieve a specific task by ID
- `PUT /api/tasks/:id`: Update a task by ID
- `DELETE /api/tasks/:id`: Delete a task by ID

## Database
The application stores task data in a SQL Server database.
